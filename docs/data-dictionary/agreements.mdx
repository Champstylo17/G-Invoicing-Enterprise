
# Agreements Table

| Column      | Type    | Description                  |
|-------------|---------|------------------------------|
| id          | int     | Primary key                  |
| title       | text    | Name of agreement            |
| agency_id   | int     | Foreign key to agency        |
| start_date  | date    | When the agreement started   |
| end_date    | date    | When the agreement ends      |
